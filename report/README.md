# Report

This folder contains the LaTeX report template and the project specification PDF.

Per the project rules (see `cmpe346_final_project.pdf`):
- The report must be compiled from the LaTeX template (`template346.tex`) — no other format accepted
- Submitted as `2_cmpe346_final.pdf`
- Must be >1000 words excluding titles and references
- Turnitin similarity must stay under 30% (including AI generation score)
- Structure: shared Introduction + one chapter per student (Methodology, Experiments & Findings, Discussion, Conclusion)

## How to compile

```bash
pdflatex template346.tex
```
