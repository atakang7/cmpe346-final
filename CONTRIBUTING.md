# Contributing Guide — CMPE346 Group 2

## Rules (non-negotiable)

- **No Jupyter Notebooks.** `.ipynb` files are blocked by `.gitignore`. Use `.py` scripts only.
- **No model weights, no datasets.** Never commit `.pt`, `.pth`, `.bin`, `.csv` files.
- **Results must be saved.** Your `main.py` must write results to `results/results.json`. See student1 as reference.
- **Use classes and separate files.** No single-file spaghetti scripts.

## Required file structure (per student folder)

```
studentN_name_topic/
├── dataset.py       # Dataset class — loads and preprocesses your dataset
├── models.py        # Model classes — LSTM, BERT, RoBERTa
├── train.py         # train() function — training loop
├── evaluate.py      # evaluate() function — returns accuracy and f1
├── main.py          # Entry point — runs all 3 models, saves results
└── results/
    └── results.json # Auto-generated by main.py — DO commit this
```

## Documentation requirements

Every file must have a module-level docstring explaining what it does.
Every class must have a one-line docstring.
Every function with non-obvious logic must have a one-line comment explaining why.

## Results format

Your `main.py` must produce `results/results.json` in this exact format:

```json
{
  "student": "Name Surname",
  "student_id": "XXXXXXXXX",
  "topic": "Your NLP Topic",
  "dataset": "dataset/name",
  "results": {
    "LSTM":     {"accuracy": 0.0000, "f1": 0.0000},
    "BERT":     {"accuracy": 0.0000, "f1": 0.0000},
    "RoBERTa":  {"accuracy": 0.0000, "f1": 0.0000}
  }
}
```

## How to run

```bash
pip install -r requirements.txt
cd studentN_name_topic
python main.py
```

Results will be saved to `results/results.json` automatically.
